Keyboard and zoom use

Use this page to follow along with the workshop.

1. Keyboard Use

Open the page below and don't use your mouse!
Use:

  • tab to progress through links & form controls
  • shift-tab to reverse
  • enter to follow links
  • cntl-f (or cmd-f) to find links

NB: Safari on OSX will only tab to form controls by default. To tab through links as well open the preferences (advanced) and select Press tab to highlight each item on a page.

Try

From help.sky.com, try getting to the article about 3-way calling, or fixing a No Satellite Signal error.

Browsers vary in the details of keyboard commands, but some combination of tab, arrows, space and enter should get you through.

2. Browser zoom

Set your browser width to 1024 or below, and open one of the pages below.
Use:

  • control + (Windows/Linux), or cmd + (Mac) to increase the zoom.
  • control - to decrease the zoom.
  • contorl 0 (zero) to reset the zoom.

Try

Compare how these two pages work differently:

  • Sky.com homepage
  • help.sky.com

NB: Safari currently has a bug where media-queries may not be triggered when zooming. Firefox, IE and (recently) Chrome work much better. The Safari bug should be fixed in the next version.

3. Screen Magnifier:

Windows: Press the windows start key and type "Magnifier".
For most versions you zoom in by pressing Windows-+,
to zoom out press Windows--.

Further instructions for Windows XP, 7 and 8.

OSX: Open system preferences > Universal Access Pane / Accessibility > seeing/zoom tab.
Set it to zoom in with scroll and ^ Control.

iOS6: System preferences > General > Accessibility > Zoom.
Enable, then double-tap with three-fingers to zoom in and out. Three-finger drag moves the screen around.

4. High contrast mode

Windows: Press the windows key and type Ease to find the Ease of access center.
Choose Set up high contrast to change your colour settings, then choose Make the computer easier to see, there is a keyboard short-cut there. IE works with this automatically.

Chrome: Has a high contrast plugin, which it suggests if it detects high-contrast mode.

OSX: Has some controls in the Accessibility section of preferences, but they are not as effective (I've not come across people using them).

To test contrast, you need the Colour Contrast Analyser (Windows app) or Colour Contrast Visualiser (Flash).

5. Screen readers

Time to take the plunge, install and try one:

  • Windows - NVDA: NVDA (Windows)

  • OSX - VoiceOver: VoiceOver (OSX)

  • iOS - VoiceOver: VoiceOver (iOS)

Turn your screen off/down and try these:

  • Sky.com
  • help.sky.com
  • accessibility.sky.com

Caveat: Just remember not to make decisions based only on your tests. Even knowing the key commands, you won't experience things in the same way as a blind or partially sighted person.

6. See the structure

Now you know what you're looking for, it's quicker to install the webdev toolbar (Firefox plugin, Chrome extension) and use these options:

  • CSS: Disable all styles
  • Images: Display alt attributes
  • Outline: Headings (tick the Show element names when outlining option)
  • Outline: Custom elements (ul, ol, li, blockquote)
  • Information: Display ARIA Roles
  • Information: View document outline
