Glossary of accessibility terms

A wide variety of words and phrases are used when talking about accessibility. This page provides descriptions and explanations for many of the most common terms you're likely to encounter.

Access technology

A hardware device or software application used by someone to help them overcome the challenges of using technology with a disability.

ActionScript

A scripting language used to develop websites and software using the Adobe Flash platform.

Adobe Acrobat

A suite of applications used to create, manipulate or read PDF files.

AJAX

Asynchronous JavaScript And XML (AJAX) is a group of interrelated technologies that can be used on the client side to create interactive web applications.

Android

An operating system from Google, widely used on mobile smart phones and other portable devices.

ARIA

Accessible Rich Internet Applications (ARIA) is a specification from the W3C that aims to make web applications more accessible to people with disabilities.

Assistive technology

See access technology.

Attribute

A characteristic of an element, usually within a markup language such as HTML or xHTML.

ATAG

The Authoring Tool Accessibility Guidelines (ATAG) from the W3C are the recognised benchmark for building accessible publishing tools for the web.

Braille

A tactile representation of spoken language consisting of cells of raised dots, used by blind and partially sighted people.

Breadcrumb

A mechanism on a web page that enables someone to understand their current location within the website and easily move back up through the page hierarchy to return to the homepage.

Captions

Spoken dialogue and sound effects from a video that are displayed in real time on the screen in text format for the benefit of deaf and hearing impaired people.

CSS

Cascading Style Sheets (CSS) are used to define the visual appearance of a website. Used in partnership with HTML or xHTML.

CMS

A Content Management System (CMS) is software or browser based application used to create, edit and publish content to a website.

Defacto

An accessible, effective and task-orientated Content Management System, created by Nomensa, for organising your website.

DOM

The Document Object Model (DOM) is a cross platform convention for representing and interacting with objects defined in HTML, xHTML and XML.

Drupal

An open source CMS used to create and maintain websites.

Elements

The individual components of an HTML document used to define content such as headings, lists, paragraphs and tables.

Flash

A rich content platform from Adobe that is used to create animated and interactive content.

Focus

The action of moving to an element on the page using a keyboard.

GZIP

A popular cross platform compression tool.

Head wand

A short rod attached to a head band, used by someone with a severe physical disability to interact with a keyboard (often with enlarged buttons) or other hardware device.

Hover

The action of moving focus to an element on the page by placing the mouse cursor over it.

HTML

HyperText Markup Language (HTML) is used to provide structure and meaning to the content of a web page. It is used in partnership with CSS (appearance) and JavaScript (behaviour).

IA

The Information Architecture (IA) is a hierarchical representation of the content contained within a website.

iOS

An operating system from Apple, used across their mobile product range including iPhone, iPod touch and iPad.

Java

A software programming language used to create stand alone and web based applications.

JavaScript

A scripting language used on the client side to add dynamic functionality to web pages.

Light box

An area of content that is visually overlaid on top of the original page, often when a button or link is activated.

Markup

The term given to languages such as HTML that are used to give structure or meaning to content.

MIME

Multipart Internet Mail Extension (MIME) is a protocol used to enable the transmission of content in multiple formats, such as binary, plain text, or audio.

Mouth wand

A similar device to a head wand, but held between the teeth instead of attached to a head band. See head wand.

Multimedia

A broad term used to define content that is animated, or which has audio/video characteristics.

PDF

Portable Document Format (PDF) is a cross platform protocol that protects the content and visual appearance of documents from being changed.

PHP

PHP HyperText Pre-processor (PHP) is a programming language used to create web applications, used in partnership with HTML.

Plug-in

A small application that can be downloaded and installed to extend the basic functionality of a browser.

Python

A programming language used to create web applications, often used in partnership with development frameworks such as Django.

Screen magnifier

A software application used by someone with partial sight to enlarge on screen information multiple times.

Screen reader

A software application used by someone with little or no usable sight to translate on screen information into synthetic speech or electronic Braille.

SEO

Search Engine Optimisation (SEO) is the process of refining a website to enhance natural search engine performance.

Sitemap

A mechanism that displays the entire navigational hierarchy of a website on a single page.

Skip link

A link that points to a different place on the same page, used to provide a shortcut for people who cannot use a mouse.

Style switcher

A widget that allows people to make choices about the way a website appears, usually in relation to text size or colour scheme.

Template

An HTML/CSS version of a web page before it is integrated into a CMS or used to create the live pages of a website.

Tool tip

A piece of text that appears over certain elements on a page when hovered over with a mouse or given focus with a keyboard.

Transcript

A text document containing the spoken dialogue, key sound effects, and visual descriptions of multimedia content.

UI

User Interface (UI) is the term used to describe the part of a website or application a person interacts with.

W3C

The World Wide Web Consortium (W3C) is the organisation that develops guidelines and standards for use on the Internet.

WAI

The Web Accessibility Initiative (WAI) is the division of the W3C responsible for accessibility guidelines and standards for the web.

WCAG

The Web Content Accessibility Guidelines (WCAG) from the W3C are the recognised benchmark for building accessible websites and measuring web accessibility.

Widget

An interactive feature or piece of dynamic content that sits within a web page.

Wireframe

A high level diagram of a web page used to describe basic layout and positioning of key areas of content.

Wordpress

An open source blog orientated CMS.

XML

eXtensible Markup Language (XML) is a meta language that allows people to define markup languages, ordinarily for use on the web.

xHTML

eXtensible HyperText Markup Language (xHTML) is an XML based markup language, closely related to HTML in characteristics and capability.
