• - off (You can turn off events that were previously applied)
  • - on (You can listen for standard and custom (non-jQuery) JS events)
  • - ready (You can detect when the document has finished loading everything except images)
  • - trigger (You can trigger standard and custom (non-jQuery) JS events)
  • - resizeend (You can know the moment (200ms) a user has finished resizing the browser)
