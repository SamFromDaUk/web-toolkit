12 Column Grid

Basic Grid

Note how classes now never have underscores '_' but only hyphens '-' instead.

The grid is made of a container 'skycom-container' which can be used to contain all your contents, or to contain individual modules.

The container is made up of up 12 columns 'skycom-1' through to 'skycom-12'.

For older browsers, the first column in a row should also have the 'alpha' class.

When the text shouldn't touch the edges of the columns (i.e. when they have a distinct background colour) use 'cushioned' class for padding.
