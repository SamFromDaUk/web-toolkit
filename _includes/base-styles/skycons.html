Skycons (retina ready)

These icons can be any size and (single) colour.

• Dont forget http://web-toolkit.global.sky.com/{{ site.version }}/fonts/skycons.css • Ensure aria-hidden is used on your skycon tags. To Create Fonts: • Open svg in inkscape, press ctrl+a to select all. • press ctrl+g to group the objects • choose Path->simplify • choose Path->union • Save font and open new font in http://petercollingridge.appspot.com/svg-editor/ • Upload and then without optimisation select 'Remove whitespace' and 'Remove All' Namespaces • Download new font into /grunt/fonts/ folder • Run grunt fonts • copy the icons JS from the skycons.css over the icons var in skycons.js • Run grunt
