Show Lightbox

To automatically bind a click event to a lightbox, give the clicky element a data-skyLightbox attribute with the id of of the lightbox. Example:

<a href="#" data-skyLightbox="#my-lightbox">Show Lightbox</a>

To manually show a lightbox, use the global toolkit show() method: 

$('#my-clickable').on('click', function(event) {
    event.preventDefault();

    // don't forget to pause any carousels, otherwise they steal tab focus!
    $('[data-function=carousel]').trigger("pause");

    // pass in the originating clicked element the lightbox may return focus returns to it when closed
    window.toolkit.lightbox.show('#lightbox', '#my-clickable');
});
$('.lightbox').skyLightbox();
