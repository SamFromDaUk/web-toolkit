Standalone Video

A wrapper for the Sky video player, in this example it's embedded onto an image. The video will expand the parent container to fit the 16/9 ratio of the video.
Ensure that href for all the social links match the 'share' urls as per the example e.g. https://twitter.com/intent/tweet?url=
Example: $(document).on('click','.play-video', function(e) { e.preventDefault(); var videoOptionsObject = { token:"Your 36 long SkyVideo Token", autoplay: true, videoId: $(this).attr('data-video-id'), freewheel: false, placeHolderImage: '' }, video, videoContainer = $(this).parents('.video'); video = new toolkit.video(videoContainer, videoOptionsObject); video.play(); }); explained: When the document element of .play-video has been clicked a function is called. Inside the callback we have the initialization of the video plugin. The video plugin accepts two parameters; Video Container (Location of the video) and Video Options Object as shown below. video = new toolkit.video(container, options);
