Skycons

These icons will look good on Retina devices and can be any size or (single) colour.

for ie7 support execute toolkit.skycons.init(); after the page has loaded.

Dont forget to include http://web-toolkit.global.sky.com/{{ site.version }}/fonts/skycons.css

Ensure aria-hidden is used on your skycon tags.

To make the icon have a colour, add the class colour to the parent.

To make the icon have a colour on hover, add the class colour-hover to the parent.

To Create Fonts: • Open svg in inkscape, press ctrl+a to select all. • press ctrl+g to group the objects • choose Path - simplify • choose Path - union • save font into /grunt/fonts/ folder • Run `grunt fonts` • copy the icons JS from the skycons.css over the icons var in skycons.js • Run `grunt`
